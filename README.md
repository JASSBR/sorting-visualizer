# Sorting Algorithm Visualizer

Visualize sorting algorithms (bubble, merge, quick, heap) with speed control and comparisons.

## Tech Stack

- HTML
- CSS
- JavaScript
- React

## Getting Started

```bash
# Clone the repo
git clone https://github.com/JASSBR/sorting-visualizer.git
cd sorting-visualizer

# Open in browser
open index.html
```
